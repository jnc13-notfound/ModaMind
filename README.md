ModaMind — Multi-Agent Fashion Brand Intelligence Platform

Turn any fashion or streetwear brand into a full strategic intelligence report — trends, ethics, consumer psychology, influencer fit, and campaign-ready content — generated by 8 specialized AI agents working in a coordinated pipeline.

Built for the Google × Kaggle Agentic AI Capstone (5-Day AI Agents: Intensive Vibe Coding Course).


Problem

Fashion and streetwear brand strategy today is fragmented. A brand needing market research, an ethics/ESG audit, consumer psychology insight, and influencer scouting typically needs four different specialists or agencies — each working in isolation, none of them talking to each other.

ModaMind consolidates this into a single automated multi-agent pipeline, with Indian market and cultural context built in from the ground up, not bolted on as an afterthought. This matters because most brand intelligence tools default to Western market assumptions (pricing tolerance, festival calendars, cultural signaling) that don't map cleanly onto Indian consumer behavior.

Why Agents

A single LLM call can summarize a brand. It can't reason across trends → positioning → ethics → psychology → influencer fit → content while catching contradictions between them — for example, flagging that a brand's "ethical craftsmanship" narrative is undermined by its own unverified leather-tanning claims. That kind of cross-domain contradiction detection is exactly what a multi-agent system with a synthesis + critic stage is built for: each agent specializes narrowly, and a downstream agent reasons across their outputs rather than in isolation.

Architecture

Eight specialized agents run in sequence, each feeding context to the next:

Trend Scout → Brand Analyst → Ethics Auditor → Consumer Psychology
→ Influencer Matcher → Synthesis → Critic → Content

| Agent | Role |
|---|---|
| Trend Scout | Surfaces emerging market and cultural signals relevant to the brand |
| Brand Analyst | Maps positioning and competitive gaps against named or auto-researched competitors |
| Ethics Auditor | Verifies sustainability/ethical claims, flags greenwashing risk with a credibility score |
| Consumer Psychology | Explains why certain campaigns or positioning will resonate with the target audience |
| Influencer Matcher | Recommends creators fit to the brand's aesthetic and audience |
| Synthesis | Cross-agent strategic brief — connects insights across all prior agents, surfaces contradictions and opportunities |
| Critic | Quality-scores the full output (0–10) before finalizing |
| Content | Generates campaign-ready copy — captions, campaign concepts, hashtags — grounded in the synthesized strategy |
On top of the pipeline, a Chat with an Agent interface lets users interrogate any individual specialist agent's findings directly, and a Report Analysis tab supports free-form Q&A grounded in the full synthesized report.

Key Features


8-agent orchestrated pipeline with a dedicated cross-agent synthesis stage
India-specific market and cultural weighting (festive calendars, pricing sensitivity, regional signaling)
Live per-agent chat — ask any specialist agent follow-up questions grounded only in its own findings
Report-wide Q&A — ask questions across the entire synthesized report
Model quota fallback — switch between Gemini models mid-session without losing pipeline state
Editorial, luxury-fashion-inspired UI (built in Gradio)


Tech Stack

Security

API keys are never hardcoded — they're loaded from environment variables via a .env file (see Setup below). No user credentials or personal data are persisted outside the active session.

Deployability

ModaMind runs locally with a single command after installing dependencies (see Setup). No external infrastructure or paid hosting is required to reproduce or judge this project.

Setup

bash
git clone https://github.com/jnc13-notfound/ModaMind
cd ModaMind
pip install -r requirements.txt

Create a .env file in the root directory:

GEMINI_API_KEY=your_key_here

Run the app:

bash
python app.py

The Gradio interface will launch locally with a shareable link printed to your terminal.
